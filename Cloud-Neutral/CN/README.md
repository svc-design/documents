# ☁️ Cloud-Neutral 系列
**—— 从云计算的自由到技术的独立**

> 云计算让我们获得了前所未有的便利，也带来了新的锁定。
> **Cloud-Neutral（云中立）** 的理念，是为了让开发者重新夺回选择权：
> 自由迁移、可退出、去依赖。
